点对点是否在一个枝干上

XYZ的上级为0，也就是说是顶级树干

X 推荐了 B 和 C

B 推荐XB1、XB2、XB3、…..XBn

C 推荐XC1、XC2、XC3、……XCn

问： XB1和XC1 和是否在一枝干上

答：某个项目告诉我不是，为什么？

XB1的分枝干包含 X、B

XC1的分枝干包含 X、C

不难看出来 他们的上级是不同的，所有不是，所以我们要实现这个需求，就是通过路径来判断

实现步骤

  1. 先找到XB1的所有父类连接成字符串
  2. 然后找到XC1的所有父类连接成字符串
  3. 比较2个字符串是否包含或被包含
